

### 客户主档 (crm_customer_profile)
| 字段含义 | 类型 | 描述 |
|----------|------|------|
| 客户ID | 字符 | 用于过滤、关联客户，是核心标识 |
| 客户名称 | 字符 | 用于搜索、筛选、排序客户 |
| 客户名称(精确) | 字符 | 用于精确匹配、聚合、排序 |
| 客户类型 | 字符 | 用于筛选客户类型（如企业/集团） |
| 客户经理员工编号 | 字符 | 用于权限范围过滤、聚合 |
| 客户经理姓名 | 字符 | 用于搜索、筛选客户经理 |
| 客户经理姓名(精确) | 字符 | 用于精确匹配客户经理 |
| 行业代码 | 字符 | 用于行业维度过滤、聚合 |
| 行业名称 | 字符 | 用于搜索行业名称 |
| 行业名称(精确) | 字符 | 用于精确匹配行业 |
| 员工规模区间 | 字符 | 用于筛选客户规模（如500-1000人） |
| 营收规模区间 | 字符 | 用于筛选营收规模（如10-50亿） |
| 地区/省份 | 字符 | 用于地理维度筛选 |
| 地区/省份(精确) | 字符 | 用于精确匹配地区 |
| 是否集团客户 | 布尔 | 用于区分集团客户/单体客户 |
| 集团ID | 字符 | 用于关联集团客户关系 |
| 集团名称 | 字符 | 用于搜索集团名称 |
| 集团名称(精确) | 字符 | 用于精确匹配集团 |
| 生命周期阶段 | 字符 | 用于客户阶段分析（如成长期/成熟期） |
| 关系评级 | 字符 | 用于客户关系质量排序（A/B/C） |
| 核心标签(多值) | 数组 | 用于标记客户特征（如"新能源"、"外贸企业"） |
| 创建时间 | 字符 | 用于时间维度过滤、排序 |
| 更新时间 | 字符 | 用于数据时效性筛选 |
